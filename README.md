# 📒 Contact Book (Python CLI App)

A simple command-line Contact Book application built in Python.

## 🔧 Features

- Add new contact
- View all contacts
- Search contact by name or phone number
- Stores contacts in a local `contacts.txt` file

## ▶️ How to Run

```bash
python contact_book.py
